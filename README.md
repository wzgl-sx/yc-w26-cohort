# YC Winter 2026 Cohort Directory

A static website showcasing Y Combinator's Winter 2026 batch companies.

## Live Site

Visit: https://wzgl-sx.github.io/yc-w26-cohort/

## About

This site features:
- 199 YC W26 companies
- Search and filter functionality
- Company details with funding info
- Responsive design

## Deployment

This site is deployed using **GitHub Pages**.

## Local Development

To run locally:

```bash
git clone https://github.com/yourusername/yc-w26-cohort.git
cd yc-w26-cohort
python3 -m http.server 8080
```

Then open http://localhost:8080
